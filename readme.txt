WP Awesome City Weather Report is a Widget that displays a specified city weather Report

== Installation ==

*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'wp-awesome-ecity-weather-report.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation. 
*  Activate the plugin.
